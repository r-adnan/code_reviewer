# Problem Statement

Junior developers often write inefficient or insecure code. 
Create an LLM-powered Python Code Review Tool that analyzes Python scripts and provides:

- Code quality feedback
- Performance suggestions
- Security warnings

**Features**

- Accept `.py` files
- Detect 
    - Bad practices
    - Code smells
    - Redundant logic
- Suggest optimized code snippets
- Rate code quality (1-10)

**Skills Covered**

- Python AST parsing
- LLM prompt design
- Static code analysis
